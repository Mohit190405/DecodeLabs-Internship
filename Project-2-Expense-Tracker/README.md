# Expense Tracker

## Overview

Expense Tracker is a simple Python application that allows users to enter multiple expenses and automatically calculates the total amount spent.

This project was developed as part of the DecodeLabs Python Programming Internship.

---

## Features

- Add multiple expenses
- Automatic total calculation
- Expense summary
- Input validation
- User-friendly interface

---
