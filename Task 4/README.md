# Multimodal Content Engine

## Overview

This project was developed as part of the DecodeLabs Generative AI Internship Program.

The system converts video transcripts into social media content using Google's Gemini AI.

## Features

- Video Transcript Analysis
- Video Summary Generation
- Viral Segment Detection
- Instagram Caption Creation
- YouTube Shorts Caption Creation
- Viral Headline Generation
- B-Roll Suggestions

## Technologies

- Python
- Streamlit
- Google Gemini
